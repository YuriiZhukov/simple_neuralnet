При запуске считываются файлы wih/who.txt для определения весов. 
Интерфейс:
1. Поле для рукописного ввода числа
2. Кнопка "Check" для проверки результата введенного числа (выводится в консоли)
3. Clear для очистки поля ввода
4. Add train data - добавляет данные текущего ввода в файл. Необходимо для обучения сети из файла  !!не реализовано до конца!!
5. Train - обучение сети текущему вводу !!необходимо указать в спинбоксе ниже то число, которому соответствует ввод перед нажатием кнопки Train!!
6. Save weights - сохраняет веса в файлы !!автоматического сохранения нет!!
